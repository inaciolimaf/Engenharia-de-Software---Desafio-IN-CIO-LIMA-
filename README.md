## Como rodar o código
Com o python intalado o primeiro passo é criar um ambiente virtual

```python -m venv venv```

Ativar o ambiente

```.\venv\Scripts\activate```

Instalar as dependências

```pip install -r requirements.txt```

E rodar o código

```python src/main.py```
